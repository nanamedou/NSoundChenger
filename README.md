# 音声調整アプリ

全体はpythonを使う  
guiはtkinterを利用している  

## サウンド再生録音

pyaudioを使う  
適宜インストールしてください  

## ファイル入出力

pydubを使う  
適宜インストールしてください  
pydubはffmpegを使うので別途インストールしてください
